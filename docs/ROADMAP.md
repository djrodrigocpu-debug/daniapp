# Roadmap proposto

## Entregue — v1.0.0: Fundação Mobile

- Fluxo demonstrativo no celular.
- Auditoria, evidências, planos e validação.
- Persistência local.

## Entregue — v1.1.0: Agenda e Prazos

- Agenda de auditorias e planos de ação.
- Visões por período e priorização de vencidos.

## Entregue — v1.2.0: Gestão Assistida

- Metas e resultados por parceiro.
- Semáforo automático.
- Diagnóstico, planos e relatório de visita.

## Entregue — v1.3.0: Publicação Web no Vercel

- React Native Web.
- Build web estático.
- Configuração automática de deploy no Vercel.

## Próxima etapa recomendada — v1.4.0: Piloto Web

- Homologação com usuários reais.
- Correções de ergonomia em desktop e celular.
- Tratamento específico de câmera e documentos no navegador.
- Testes automatizados dos fluxos críticos.

## Etapa estrutural — v2.0.0: Plataforma Corporativa

- API e banco central.
- Autenticação corporativa.
- Sincronização e modo offline robusto.
- Armazenamento seguro de evidências.
- Perfis, permissões, auditoria e LGPD.
