# Roadmap proposto

## Versão 1.0 — Fundação Mobile

- Fluxo completo demonstrativo no celular.
- Dados locais.
- Auditoria, evidências, planos e validação.

## Versão 1.1 — Refinamento de campo

- Ajustes a partir do piloto.
- Melhorias de usabilidade e desempenho.
- Filtros adicionais.
- Resumo por pilar.
- Registro da observação da devolução no checklist.
- Exportação local de uma auditoria em PDF.

## Versão 1.2 — Sincronização central

- API e banco central.
- Login corporativo.
- Sincronização de usuários, hierarquia e operações.
- Repositório corporativo de evidências.
- Trilhas de auditoria de servidor.

## Versão 1.3 — Gestão e alertas

- Notificações de vencimento.
- Alertas de reincidência.
- Agenda de auditorias.
- Indicadores evolutivos.
- Painel regional conectado ao Power BI.

## Versão 1.4 — Automação de indicadores

- Integração com fontes oficiais.
- Pré-preenchimento dos KPIs.
- Regras de semáforo parametrizadas no servidor.
- Catálogo administrável.

## Versão 2.0 — Plataforma corporativa

- Ambiente produtivo escalável.
- Operação offline com sincronização robusta.
- Governança de dados e LGPD completa.
- Gestão de perfis e permissões corporativas.
- Observabilidade, suporte e continuidade de negócio.
